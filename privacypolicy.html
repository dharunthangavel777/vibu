This Privacy Policy outlines the information concerning the collection, use, and protection of user data through the Company's Website www.vibuh.com ("the Website") and Vibhu Customer ("the Application"). By using the Website and the Application, you agree to the terms of this Privacy Policy.

The Company's Website and the Application collects the following user data for the specified purposes:

All user data collected by the Website and the Application will be stored on an on-premise secure server maintained by the Company.

The user data collected through the Website and the Application is used for the following purposes:

By using the website, you consent to the collection of cookies and cache for the purpose of improving user experience, optimizing performance, and providing personalized services. The user has the option to manage cookie preferences through the website.

The Company ensures that any cookies and cache collected are used solely for the specified purposes and are not shared with any third parties for advertising or marketing purposes.
Data Sharing
The Company is committed to safeguarding the privacy of its users. User data will not be shared with any third party for any purposes, except as required by law or with explicit user consent.
Data Retention
User data will be securely stored on the on-premise server for as long as necessary to fulfil the purposes outlined in this Privacy Policy, or as required by law. After this period, the data will be securely deleted.
